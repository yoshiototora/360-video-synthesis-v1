# 深度推定を用いた360°映像の高品質合成システム
(High-quality 360° Video Synthesis System using Depth Estimation)

## 📝 概要
360°映像の合成において発生する不自然な境界を解消するため、**深度推定を用いた前景分離と動的閾値処理を組み合わせた映像合成手法**を提案・実装しました。

従来の画素ベースの手法では物体の輪郭と無関係な位置で映像が切断されてしまう問題がありましたが、本手法により、物体単位での自然な境界での映像合成を実現しています。

*(※ デモ用の比較画像・動画は現在準備中です)*

## 🚨 背景・課題
360°映像（正距円筒図法）において、複数カメラの映像を合成する際に以下の問題が発生します。

- カメラ間の視差および露出差
- 物体の途中での不自然な切断
- 直線的で目立つつなぎ目

特に従来手法（固定領域による合成）では、合成境界が物体の輪郭と一致しないため、視覚的な違和感が大きく残るという課題がありました。

## 💡 提案手法
本研究では、以下の3つの技術アプローチを組み合わせて課題を解決しました。

1. **深度推定による前景分離**
   単眼画像から深度を推定し、前景と背景を分離。これにより、画素単位ではなく「物体単位」での自然な合成を実現。
2. **動的閾値の導入**
   シーンごとの深度分布に応じて合成の閾値を動的に決定。屋内外の環境変化や多様なシーンに柔軟に適応可能。
3. **多段階フェード処理**
   分離した境界部分にフェード処理を適用し、不連続なつなぎ目を視覚的に緩和。

## ✨ 工夫点（差別化ポイント）
- **環境適応性の向上**: 固定閾値に依存せず、動的閾値を採用したことで汎用性が向上。
- **視覚的品質の追求**: 前景分離とフェード処理のハイブリッドにより、境界の自然さを劇的に改善。
- **アクセシビリティの確保**: GPU環境だけでなく、**CPU環境でも動作するよう処理を最適化**。

## 📊 結果
従来手法と比較し、以下の点で明確な改善を確認しました。
- 境界の自然さが向上
- 物体の不自然な切断が大幅に減少
- 全体的な視覚的品質の改善（主観評価においても高評価を獲得）
